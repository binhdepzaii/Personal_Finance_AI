# Personal Finance AI

Ứng dụng quản lý tài chính cá nhân có trợ lý AI.

## Công nghệ sử dụng

- Frontend: HTML, CSS, JavaScript, TailwindCSS, Chart.js
- Backend: Python FastAPI
- Database: SQLite
- AI: Gemini API kết hợp rule-based fallback

## Chức năng chính

- Đăng ký, đăng nhập người dùng
- Quản lý ví tiền
- Thêm, xem, xóa giao dịch
- Tính tổng thu, tổng chi, số dư, tỷ lệ tiết kiệm
- Biểu đồ tài chính
- Trợ lý AI nhập giao dịch bằng ngôn ngữ tự nhiên
- AI tách nhiều giao dịch trong một câu
- Hỏi đáp tài chính cá nhân
- Báo cáo tài chính
- Xuất CSV
- In hoặc lưu PDF
